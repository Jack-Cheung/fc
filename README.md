# fc
fork 自 EOSIO/fc,一个很棒的C++库，可以作为标准库来使用，
添加了 MSVC目录，内含vc++的项目文件（x64 Debug），使项目可以用MSVC编译。
不过要先安装 Boost 和 openssl，
我分别安装在D:\Dev\boost_1_69_0 和 D:\Dev\OpenSSL-Win64,
你可以安装在自己指定的目录，但是要在MSVC project file中指定这两个位置
